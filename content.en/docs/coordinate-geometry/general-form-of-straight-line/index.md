## 5.6 General Form of a Straight Line

The linear equation (first degree polynomial in two variables {{< katex >}} x {{< /katex >}}
 and {{< katex >}} y {{< /katex >}}):

{{< katex >}}
ax + by + c = 0
{{< /katex >}}

(where {{< katex >}}
a, b{{< /katex >}}
 and {{< katex >}}
c
{{< /katex >}} are real numbers such that at least one of 
{{< katex >}}a, b
{{< /katex >}} is non-zero) always represents a straight line. This is the **general form of a straight line**.

Now, let us find out the equations of a straight line in the following cases:
- (i) Parallel to {{< katex >}}
ax + by + c = 0
{{< /katex >}}
- (ii) Perpendicular to {{< katex >}}
ax + by + c = 0
{{< /katex >}}

### 5.6.1 Equation of a line parallel to the line
{{< katex >}}
ax + by + c = 0
{{< /katex >}}

The equation of all lines parallel to the line {{< katex >}}
ax + by + c = 0
{{< /katex >}} can be put in the form:

{{< katex >}}
ax + by + k = 0
{{< /katex >}}

for different values of {{< katex >}}k{{< /katex >}}.

### 5.6.2 Equation of a line perpendicular to the line 
{{< katex >}}
ax + by + c = 0
{{< /katex >}}

The equation of all lines perpendicular to the line {{< katex >}}
ax + by + c = 0
{{< /katex >}} can be written as:

{{< katex >}}
bx - ay + k = 0
{{< /katex >}}

for different values of {{< katex >}}
k
{{< /katex >}}.

Two straight lines {{< katex >}}
a_1x + b_1y + c_1 = 0{{< /katex >}}
 and 
{{< katex >}}a_2x + b_2y + c_2 = 0
{{< /katex >}} where the coefficients are non-zero, are:

- (i) **parallel** if and only if {{< katex >}}
\frac{a_1}{a_2} = \frac{b_1}{b_2}{{< /katex >}}
; That is 
{{< katex >}}a_1b_2 - a_2b_1 = 0
{{< /katex >}}
- (ii) **perpendicular** if and only if {{< katex >}}
a_1a_2 + b_1b_2 = 0
{{< /katex >}}

### 5.6.3 Slope of a straight line

The general form of the equation of a straight line is {{< katex >}}
ax + by + c = 0{{< /katex >}}
 (at least one of 
{{< katex >}}a, b
{{< /katex >}} is non-zero).

Coefficient of {{< katex >}}
x = a{{< /katex >}}
, coefficient of {{< katex >}}
y = b
{{< /katex >}}, constant term 
{{< katex >}}= c
{{< /katex >}}.

The above equation can be rewritten as:

{{< katex >}}
by = -ax - c
{{< /katex >}}

{{< katex >}}
y = -\frac{a}{b}x - \frac{c}{b}, \quad \text{if } b \neq 0 \quad \text{...(1)}
{{< /katex >}}

Comparing (1) with the form {{< katex >}}
y = mx + l
{{< /katex >}}:

We get:
- **Slope** {{< katex >}}
m = -\frac{a}{b} = -\frac{\text{coefficient of } x}{\text{coefficient of } y}
{{< /katex >}}
- **y-intercept** {{< katex >}}
l = -\frac{c}{b} = -\frac{\text{constant term}}{\text{coefficient of } y}
{{< /katex >}}

### Example 5.30

Find the slope of the straight line {{< katex >}}

{{< /katex >}}.

**Solution:** Given {{< katex >}}
6x + 8y + 7 = 0
{{< /katex >}}

{{< katex >}}
\text{Slope } m = \frac{-\text{coefficient of } x}{\text{coefficient of } y} = -\frac{6}{8} = -\frac{3}{4}
{{< /katex >}}

Therefore, the slope of the straight line is {{< katex >}}
-\frac{3}{4}
{{< /katex >}}.

### Example 5.31

Find the slope of the line which is:
(i) Parallel to {{< katex >}}
3x - 7y = 11
{{< /katex >}}
(ii) Perpendicular to {{< katex >}}
2x - 3y + 8 = 0
{{< /katex >}}

**Solution:**

(i) Given straight line is {{< katex >}}
3x - 7y = 11 \Rightarrow 3x - 7y - 11 = 0
{{< /katex >}}

Slope {{< katex >}}
m = -\frac{3}{-7} = \frac{3}{7}
{{< /katex >}}

Since parallel lines have same slopes, slope of any line parallel to {{< katex >}}
3x - 7y = 11{{< /katex >}}
 is 
{{< katex >}}\frac{3}{7}
{{< /katex >}}.

(ii) Given straight line is {{< katex >}}
2x - 3y + 8 = 0
{{< /katex >}}

Slope {{< katex >}}
m = \frac{-2}{-3} = \frac{2}{3}
{{< /katex >}}

Since product of slopes is {{< katex >}}
-1{{< /katex >}}
 for perpendicular lines, slope of any line perpendicular to {{< katex >}}
2x - 3y + 8 = 0
{{< /katex >}} is 
{{< katex >}}\frac{-1}{\frac{2}{3}} = -\frac{3}{2}
{{< /katex >}}.

### Example 5.32

Show that the straight lines {{< katex >}}
2x + 3y - 8 = 0{{< /katex >}}
 and 
{{< katex >}}4x + 6y + 18 = 0
{{< /katex >}} are parallel.

**Solution:** Slope of the straight line {{< katex >}}
2x + 3y - 8 = 0
{{< /katex >}} is:

{{< katex >}}
m_1 = \frac{-\text{coefficient of } x}{\text{coefficient of } y} = \frac{-2}{3}
{{< /katex >}}

Slope of the straight line {{< katex >}}
4x + 6y + 18 = 0
{{< /katex >}} is:

{{< katex >}}
m_2 = \frac{-4}{6} = \frac{-2}{3}
{{< /katex >}}

Here, {{< katex >}}
m_1 = m_2
{{< /katex >}}. That is, slopes are equal. Hence, the two straight lines are parallel.

**Aliter:**

{{< katex >}}
a_1 = 2, b_1 = 3{{< /katex >}}
; 
{{< katex >}}a_2 = 4, b_2 = 6
{{< /katex >}}

{{< katex >}}
\frac{a_1}{a_2} = \frac{2}{4} = \frac{1}{2}
{{< /katex >}}

{{< katex >}}
\frac{b_1}{b_2} = \frac{3}{6} = \frac{1}{2}
{{< /katex >}}

Therefore, {{< katex >}}
\frac{a_1}{a_2} = \frac{b_1}{b_2}
{{< /katex >}}. Hence the lines are parallel.

### Example 5.33

Show that the straight lines {{< katex >}}
x - 2y + 3 = 0{{< /katex >}}
 and 
{{< katex >}}6x + 3y + 8 = 0
{{< /katex >}} are perpendicular.

**Solution:** Slope of the straight line {{< katex >}}
x - 2y + 3 = 0
{{< /katex >}} is:

{{< katex >}}
m_1 = \frac{-1}{-2} = \frac{1}{2}
{{< /katex >}}

Slope of the straight line {{< katex >}}
6x + 3y + 8 = 0
{{< /katex >}} is:

{{< katex >}}
m_2 = \frac{-6}{3} = -2
{{< /katex >}}

Now, {{< katex >}}
m_1 \times m_2 = \frac{1}{2} \times (-2) = -1
{{< /katex >}}

Hence, the two straight lines are perpendicular.

**Aliter:**

{{< katex >}}
a_1 = 1, b_1 = -2{{< /katex >}}
; 
{{< katex >}}a_2 = 6, b_2 = 3
{{< /katex >}}

{{< katex >}}
a_1a_2 + b_1b_2 = 6 - 6 = 0
{{< /katex >}}

The lines are perpendicular.

## Points to Remember

- The area of a triangle formed by the points {{< katex >}}
(x_1, y_1){{< /katex >}}
, {{< katex >}}
(x_2, y_2)

{{< /katex >}}

- Three points {{< katex >}}
A(x_1, y_1){{< /katex >}}
, {{< katex >}}
B(x_2, y_2)
{{< /katex >}} and 
{{< katex >}}C(x_3, y_3)
{{< /katex >}} are collinear if and only if:
  - (i) Area of {{< katex >}}
\triangle ABC = 0{{< /katex >}}
 or 
{{< katex >}}x_1y_2 + x_2y_3 + x_3y_1 = x_2y_1 + x_3y_2 + x_1y_3

  - (ii) Slope of {{< katex >}}
AB{{< /katex >}}
 = slope of {{< katex >}}
BC
{{< /katex >}} or slope of 
{{< katex >}}AC
{{< /katex >}}

, {{< katex >}}
(x_2, y_2){{< /katex >}}
, 
{{< katex >}}(x_3, y_3)

## Equation of straight line in various forms

| Form | Name |
|------|------|
| {{< katex >}}
ax + by + c = 0
{{< /katex >}} | General form |
| {{< katex >}}
y = mx + c
{{< /katex >}} | Slope-intercept |
| {{< katex >}}
y - y_1 = m(x - x_1)
{{< /katex >}} | Point-slope form |
| {{< katex >}}
\frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1}
{{< /katex >}} | Two point form |
| {{< katex >}}
\frac{x}{a} + \frac{y}{b} = 1
{{< /katex >}} | Intercept form |
| {{< katex >}}
x = c
{{< /katex >}} | Parallel to Y-axis |
| {{< katex >}}
y = b
{{< /katex >}} | Parallel to X-axis |

- Two straight lines are **parallel** if and only if their slopes are equal.
- Two straight lines with well-defined slopes {{< katex >}}
m_1, m_2{{< /katex >}}
 are **perpendicular** if and only if 
{{< katex >}}m_1 \times m_2 = -1
{{< /katex >}}.
