## 5.5 Straight Line

Any first degree equation in two variables {{< katex >}}
x{{< /katex >}}
 and 
{{< katex >}}y
{{< /katex >}} of the form:

{{< katex >}}
ax + by + c = 0 \quad \text{...(1)}
{{< /katex >}}

where {{< katex >}}
a, b, c{{< /katex >}}
 are real numbers and at least one of 
{{< katex >}}a, b
{{< /katex >}} is non-zero is called **"Straight line"** in XY plane.
![](30.png)

### 5.5.1 Equation of coordinate axes

The X-axis and Y-axis together are called **coordinate axes**.

The x-coordinate of every point on {{< katex >}}
OY{{< /katex >}}
 (Y-axis) is {{< katex >}}
0{{< /katex >}}
. Therefore equation of 
{{< katex >}}OY
{{< /katex >}} (Y-axis) is 
{{< katex >}}x = 0
{{< /katex >}}.
![](31.png)

The y-coordinate of every point on {{< katex >}}
OX{{< /katex >}}
 (X-axis) is {{< katex >}}
0{{< /katex >}}
. Therefore the equation of 
{{< katex >}}OX
{{< /katex >}} (X-axis) is 
{{< katex >}}y = 0
{{< /katex >}}.
![](32.png)

### 5.5.2 Equation of a straight line parallel to X axis

Let {{< katex >}}
AB{{< /katex >}}
 be a straight line parallel to X-axis, which is at a distance '{{< katex >}}
b{{< /katex >}}
'. Then y-coordinate of every point on '
{{< katex >}}AB
{{< /katex >}}' is '
{{< katex >}}b
{{< /katex >}}'.

Therefore, the equation of {{< katex >}}
AB{{< /katex >}}
 is 
{{< katex >}}y = b
{{< /katex >}}.

![](33.png)

> **Note:**
> - If {{< katex >}}
b > 0{{< /katex >}}
, then the line 
{{< katex >}}y = b
{{< /katex >}} lies above the X-axis
> - If {{< katex >}}
b < 0{{< /katex >}}
, then the line 
{{< katex >}}y = b
{{< /katex >}} lies below the X-axis
> - If {{< katex >}}
b = 0{{< /katex >}}
, then the line 
{{< katex >}}y = b
{{< /katex >}} is the X-axis itself.

### 5.5.3 Equation of a Straight line parallel to the Y axis

Let {{< katex >}}
CD{{< /katex >}}
 be a straight line parallel to Y-axis, which is at a distance '{{< katex >}}
c{{< /katex >}}
'. Then x-coordinate of every point on {{< katex >}}
CD{{< /katex >}}
 is '
{{< katex >}}c
{{< /katex >}}'. The equation of 
{{< katex >}}CD
{{< /katex >}} is 
{{< katex >}}x = c
{{< /katex >}}.

![](34.png)

- If {{< katex >}}
c > 0{{< /katex >}}
, then the line 
{{< katex >}}x = c
{{< /katex >}} lies right to the side of the Y-axis
- If {{< katex >}}
c < 0{{< /katex >}}
, then the line 
{{< katex >}}x = c
{{< /katex >}} lies left to the side of the Y-axis
- If {{< katex >}}
c = 0{{< /katex >}}
, then the line 
{{< katex >}}x = c
{{< /katex >}} is the Y-axis itself.

### Example 5.17

Find the equation of a straight line passing through {{< katex >}}
(5,7)
{{< /katex >}} and is (i) parallel to X-axis (ii) parallel to Y-axis.

**Solution:**

(i) The equation of any straight line parallel to X-axis is {{< katex >}}

. Since it passes through {{< katex >}}
(5,7){{< /katex >}}
, 
{{< katex >}}b = 7

{{< /katex >}}.

x = c{{< /katex >}}
. Since it passes through {{< katex >}}
(5,7){{< /katex >}}

{{< katex >}}c = 5

{{< /katex >}}.

### 5.5.4 Slope-Intercept Form

Every straight line that is not vertical will cut the Y-axis at a single point. The y-coordinate of this point is called **y-intercept** of the line.

A line with slope {{< katex >}}
m{{< /katex >}}
 and y-intercept 
{{< katex >}}c
{{< /katex >}} can be expressed through the equation:

{{< katex >}}
y = mx + c
{{< /katex >}}

We call this equation as the **slope-intercept form** of the equation of a line.

> **KNOW?**
> - If a line with slope {{< katex >}}
m{{< /katex >}}
, {{< katex >}}
m \neq 0{{< /katex >}}
 makes x-intercept 
{{< katex >}}d
{{< /katex >}}, then the equation of the straight line is 
{{< katex >}}y = m(x-d)
{{< /katex >}}.
> - {{< katex >}}
y = mx{{< /katex >}}
 represents equation of a straight line with slope 
{{< katex >}}m
{{< /katex >}} and passing through the origin.

{{< /katex >}}

### 5.5.5 Point-Slope form
![](36.png)

Here we will find the equation of a straight line passing through a given point {{< katex >}}
A(x_1, y_1){{< /katex >}}
 and having the slope 
{{< katex >}}m
{{< /katex >}}.

Let {{< katex >}}
P(x,y){{< /katex >}}
 be any point other than {{< katex >}}
A{{< /katex >}}
 on the given line. Slope of the line joining 
{{< katex >}}A(x_1, y_1)
{{< /katex >}} and 
{{< katex >}}P(x,y)
{{< /katex >}} is given by:

{{< katex >}}
m = \tan\theta = \frac{y - y_1}{x - x_1}
{{< /katex >}}

Therefore, the equation of the required line is:

{{< katex >}}
y - y_1 = m(x - x_1) \quad \text{(Point-slope form)}
{{< /katex >}}

{{< katex >}}

{{< /katex >}}

{{< katex >}}

{{< /katex >}}

### 5.5.6 Two Point form

Let {{< katex >}}
A(x_1, y_1){{< /katex >}}
 and {{< katex >}}
B(x_2, y_2){{< /katex >}}
 be two given distinct points. Slope of the straight line passing through these points is given by 
{{< katex >}}m = \frac{y_2 - y_1}{x_2 - x_1}
{{< /katex >}}, 
{{< katex >}}(x_1 \neq x_2)
{{< /katex >}}.

From the equation of the straight line in point-slope form, we get:

{{< katex >}}
y - y_1 = \frac{y_2 - y_1}{x_2 - x_1}(x - x_1)
{{< /katex >}}

Hence,

{{< katex >}}
\frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1}
{{< /katex >}}

is the equation of the line in **two-point form**.

Substituting the points we get:

{{< katex >}}

{{< katex >}}

{{< /katex >}}

{{< katex >}}

{{< /katex >}}

{{< katex >}}

{{< katex >}}

### 5.5.7 Intercept Form

We will find the equation of a line whose intercepts are {{< katex >}}
a{{< /katex >}}
 and 
{{< katex >}}b
{{< /katex >}} on the coordinate axes respectively.
![](39.png)

Let {{< katex >}}
PQ{{< /katex >}}
 be a line meeting X-axis at {{< katex >}}
A{{< /katex >}}
 and Y-axis at {{< katex >}}
B{{< /katex >}}
. Let {{< katex >}}
OA = a{{< /katex >}}
, {{< katex >}}
OB = b{{< /katex >}}
. Then the coordinates of {{< katex >}}
A
{{< /katex >}} and 
{{< katex >}}B
{{< /katex >}} are 
{{< katex >}}(a, 0)
{{< /katex >}} and 
{{< katex >}}(0, b)
{{< /katex >}} respectively. Therefore, the equation of the line joining 
{{< katex >}}A
{{< /katex >}} and 
{{< katex >}}B
{{< /katex >}} is:

{{< katex >}}
\frac{y - 0}{b - 0} = \frac{x - a}{0 - a}
{{< /katex >}}

{{< katex >}}
\frac{y}{b} = \frac{x - a}{-a}
{{< /katex >}}

{{< katex >}}
\frac{y}{b} = -\frac{x}{a} + 1
{{< /katex >}}

Hence,

{{< katex >}}
\frac{x}{a} + \frac{y}{b} = 1 \quad \text{(Intercept form of a line)}
{{< /katex >}}
