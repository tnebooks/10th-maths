# 4.5 Circles and Tangents

In our day-to-day real life situations, we have seen two lines intersect at a point or do not intersect in a plane. For example, two parallel lines in a railway track, do not intersect. Whereas, grills in a window intersect.

![](1.png)


Similarly what happens when a curve and a line is given in a plane? The curve may be parabola, circle or any general curve.

We may get three situations:

![](2.png)


![](3.png)


## Definition

If a line touches the given circle at only one point, then it is called **tangent** to the circle.

---

## Real Life Examples of Tangents to Circles

1. When a cycle moves along a road, then the road becomes the tangent at each point when the wheels rolls on it.

![](4.png)


2. When a stone is tied at one end of a string and is rotated from the other end, then the stone will describe a circle. If we suddenly stop the motion, the stone will go in a direction tangential to the circular motion.

![](5.png)

## Theorem 6: Alternate Segment Theorem

**Statement:** If a line touches a circle and from the point of contact a chord is drawn, the angles between the tangent and the chord are respectively equal to the angles in the corresponding alternate segments.


![](13.png)


**Proof:**

**Given:** A circle with centre at  {{< katex  >}}
O {{< /katex  >}}
, tangent  {{< katex  >}}
AB {{< /katex  >}}
 touches the circle at  {{< katex  >}}
P {{< /katex  >}}
 and  {{< katex  >}}
PQ
{{< /katex >}} is a chord. 
{{< katex >}}S
{{< /katex >}} and 
{{< katex >}}T
{{< /katex >}} are two points on the circle in the opposite sides of chord 
{{< katex >}}PQ
{{< /katex >}}.

**To prove:** 
- (i)  {{< katex  >}}
\angle QPB = \angle PSQ
{{< /katex >}} 
- (ii)  {{< katex  >}}
\angle QPA = \angle PTQ
{{< /katex >}}

**Construction:** Draw the diameter  {{< katex  >}}
POR {{< /katex  >}}
. Draw  {{< katex  >}}
QR {{< /katex  >}}
, 
{{< katex >}}QS
{{< /katex >}} and 
{{< katex >}}PS
{{< /katex >}}.

| No. | Statement | Reason |
|:---|:---|:---|
| 1. |  {{< katex  >}}\angle RPB = 90° {{< /katex  >}} | Diameter  {{< katex  >}}RP{{< /katex >}} is perpendicular to tangent {{< katex >}}AB{{< /katex >}}. || | Now,  {{< katex  >}}\angle RPQ + \angle QPB = 90°{{< /katex >}} ... (1) | |
| 2. | In  {{< katex  >}}\triangle RPQ {{< /katex  >}}, {{< katex >}}\angle PQR = 90°{{< /katex >}} ... (2) | Angle in a semicircle is 90°. |
| 3. |  {{< katex  >}}\angle QRP + \angle RPQ = 90°{{< /katex >}} ... (3) | In a right angled triangle, sum of the two acute angles is 90°. |
| 4. |  {{< katex  >}}\angle RPQ + \angle QPB = \angle QRP + \angle RPQ{{< /katex >}} | From (1) and (3). || |  {{< katex  >}}\angle QPB = \angle QRP{{< /katex >}} ... (4) | |
| 5. |  {{< katex  >}}\angle QRP = \angle PSQ{{< /katex >}} ... (5) | Angles in the same segment are equal. |
| 6. |  {{< katex  >}}\angle QPB = \angle PSQ{{< /katex >}} ... (6) | From(4) and (5); Hence (i) is proved. |
| 7. |  {{< katex  >}}\angle QPB + \angle QPA = 180°{{< /katex >}} ... (7) | Linear pair of angles. |
| 8. |  {{< katex  >}}\angle PSQ + \angle PTQ = 180°{{< /katex >}} ... (8) | Sum of opposite angles of a cyclic quadrilateral is 180°. |
| 9. |  {{< katex  >}}\angle QPB + \angle QPA = \angle PSQ + \angle PTQ{{< /katex >}} | From (7) and (8). |
| 10. |  {{< katex  >}}
\angle QPB + \angle QPA = \angle QPB + \angle PTQ {{< /katex  >}} | {{< katex >}}\angle QPB = \angle PSQ{{< /katex >}} from (6) |
| 11. |  {{< katex  >}}\angle QPA = \angle PTQ{{< /katex >}} | Hence (ii) is proved. |

This completes the proof.
